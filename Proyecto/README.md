1. Clona el repositorio:


2. Ingresa al directorio del proyecto:
   ```bash

   ```

3. Instala las dependencias:
   ```bash

   npm start
   ```

Esto abrirá la aplicación en tu navegador predeterminado. Puedes verla en [http://localhost:3000](http://localhost:3000).
